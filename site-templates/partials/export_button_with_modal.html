Raw Data Export

The raw data export is a list of all solved tasks in csv file format. It includes for each task:

  • user id - a users unique identifier
  • task - unique identifier of the task
  • loading time - how long a user was waiting for a task to load in milliseconds
  • processing time - how long a user was processing a task in milliseconds (time between load and saving a task)
